# 影像處理課程 Image Processing Course
 
本課程以 OpenCV 為主要工具，涵蓋數位影像處理的核心理論與實作，從基礎操作到進階應用循序漸進。
 
---
 
## 課程目錄
 
| 週次 | 主題 |
|------|------|
| W1  | OpenCV 介紹 & 基本操作 |
| W2  | 數位影像基礎 |
| W3  | 幾何變換 |
| W4  | 強度轉換 & 直方圖 |
| W5  | 影像捲積 |
| W6  | 平滑濾波器（空間域） |
| W7  | 平滑濾波器 2 |
| W8  | 銳化濾波器 |
| W9  | 頻率域基本操作 |
| W10 | 影像復原與重建 |
| W11 | 色彩影像基礎 |
| W12 | 小波與其他轉換 |
| W13 | 形態學影像處理 |
| W14 | 影像分割與實務應用 |
 
---
 
## 環境需求

```bash
pip install opencv-python numpy matplotlib
```
